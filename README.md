# 🚀 English AI Booster: Aprenda Inglês com Engenharia de Prompts

Este repositório apresenta um framework de **Engenharia de Prompts** projetado para transformar IAs (ChatGPT, Claude, Gemini) em tutores particulares de inglês para quem tem pouco tempo.

## 🛠️ Técnicas Utilizadas
- **Persona Acting**: IA assume papel de professor ou nativo.
- **Few-Shot Prompting**: Exemplos para guiar a resposta.
- **Feedback Loops**: Correções gramaticais em tempo real.

## 📂 Prompts Prontos (Copie e Use)

### 1. Simulador de Trabalho (10 min)
**Prompt:** "Aja como meu chefe americano. Inicie uma conversa curta sobre o status de um projeto. Corrija meu inglês após cada frase minha e sugira termos mais profissionais."

### 2. Pílula de Vocabulário (5 min)
**Prompt:** "Sou da área de [Sua Área]. Liste 3 termos técnicos em inglês, seus significados e uma frase de exemplo para cada. Formate em uma tabela."
